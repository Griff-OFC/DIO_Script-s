# Repositório — Aulas da DIO

**Propósito**

Este repositório foi criado para armazenar e organizar **todos os arquivos e scripts apresentados nas aulas da DIO sobre cibersegurança**. Aqui você encontrará exemplos práticos, scripts de demonstração, anotações e pequenos utilitários utilizados durante as aulas — tudo com o objetivo de estudo e prática em ambiente controlado.

> ⚠️ **Importante:** Os scripts deste repositório (por exemplo, ferramentas de brute-force, varredura de portas, automações) são fornecidos **apenas para fins educacionais**. O uso em sistemas alheios sem autorização explícita é ilegal e antiético. Utilize-os somente em laboratórios, máquinas de teste ou ambientes onde você tem permissão para realizar testes.
